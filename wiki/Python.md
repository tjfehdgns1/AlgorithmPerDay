### Dictionary

##### Built-in functions & methods
###### .fromkeys()
```python
keys = ['a', 'b', 'c']
initial_value = 0
dic = dict.fromkeys(keys, initial_value)
print(dic) # {'a': 0, 'b': 0, 'c': 0}
```
```python
string = "hello"
dic = dict.fromkeys(string)
print(dic)# {'h': None, 'e': None, 'l': None, 'o': None}
```
